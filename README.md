# Sicherungskarte
Sicherungskarte für 24V-Systeme, zum Absichern von Lasten und Stromkreisen.

---
## 🔧 Gedachte Verwendung
An die oberen Schraubklemmen wird die Versorgungsspannung angeschlossen.  
An die unteren Schraubklemmen wird die Last bzw. der Stromkreis der abgesichert werden soll, angeschlossen.  

Die **maximale** Sicherungsgröße beträgt **10A!**  

---

## 📏 Technische Daten
- Versorgungsspannung: 24V (+-20%)
- Laststrom: 10A max
- PCB Maße: 56 mm x 52 mm
